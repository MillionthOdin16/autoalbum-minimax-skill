# Style Catalog Research Notes

These notes summarize public sources used to build the style catalog. Keep this file as a source index for later expansion.

## MiniMax-specific sources

- MiniMax direct Music Generation API docs: `music-2.6`, prompt and lyrics parameters, stream/output_format behavior, structure tags, cover inputs, response fields.
  - https://platform.minimax.io/docs/api-reference/music-generation
- MiniMax Music Generation Guide: cover workflow and cover preprocess behavior.
  - https://platform.minimax.io/docs/guides/music-generation
- MiniMax official prompt guide: vivid English sentences, not comma-separated tags; mood/genre/vocal/theme/production prompt format.
  - https://github.com/MiniMax-AI/skills/blob/main/skills/minimax-music-gen/references/prompt_guide.md?plain=1
- MiniMax music skill / CLI documentation: structured fields such as genre, mood, vocals, instruments, BPM, key, structure, references, avoid.
  - https://github.com/MiniMax-AI/skills/blob/main/skills/minimax-music-gen/SKILL.md?plain=1
- MiniMax Music 2.6 public product page: audio quality, formats, structure tags, cover and lyrics optimizer concepts.
  - https://www.minimax-music.com/minimax-music-2-6

## Hip-hop / rap sources

- Memphis rap characteristics: low-budget/lo-fi soundscape, TR-808, minimal synth melodies, triplet/double-time flows, cowbell, syncopated rhythms, sub-bass, sharp snares.
  - https://en.wikipedia.org/wiki/Memphis_rap
- Memphis underground rap overview: dark lo-fi production, triplet flows, repetitive hooks.
  - https://felixonline.co.uk/articles/2019-10-25-foundations-hip-hop-memphis-underground/
- Grammy guide to hip-hop subgenres: drill characterized by dark/menacing energy, distorted 808 basslines, dark synths, trap-style hats.
  - https://grammy.com/news/11-subgenres-of-hip-hop-jersey-club-drill-trap-examples-videos
- Boom bap characteristics: kick/snare “boom bap,” minimal instrumentation, sample-based loops, swing, Akai MPC/SP1200 context.
  - https://en.wikipedia.org/wiki/Boom_bap
- Trap production guides: tuned 808s, fast/slow hi-hat alternation, triplets, clipped kicks/snappy snares.
  - https://mixedinkey.com/captain-plugins/wiki/trap-beat/
  - https://www.productionmusiclive.com/blogs/news/trap-beat-guide-drums-essential-tips-for-making-trap-drum-patterns

## Latin/global sources

- Dembow rhythm: 3+3+2/tresillo cross-rhythm, reggaeton core percussion.
  - https://en.wikipedia.org/wiki/Dembow_beat
- Berklee dembow explanation: Dominican dembow as harder/faster derivative; reggaeton roots.
  - https://www.berklee.edu/berklee-now/news/what-is-dembow-tracing-the-roots-of-a-global-phenomenon
- Reggaeton production: dembow variant, 85–100 BPM, singing/rap/instrumentation backdrop.
  - https://www.musicradar.com/news/beat-building-how-to-produce-a-reggaeton-beat
- Amapiano production: log drum as defining element, 105–115 BPM range.
  - https://articles.roland.com/production-hacks-creating-amapiano-tracks/
  - https://create.routenote.com/blog/beatmakers-guide-how-to-make-an-amapiano-beat/

## Expansion instructions

When adding a new artist profile:

1. Research at least three sources when possible: one overview, one critical review/analysis, and one production/songwriting source.
2. Extract concrete musical traits, not fan adjectives.
3. Create era-specific lanes if the artist changed style significantly.
4. Add a MiniMax prompt fragment.
5. Add failure modes and album use cases.
6. Add source URLs here.


## Style catalog refinement notes
- Style profiles must encode rhythm, bass, vocal delivery, hook mechanics, and mix space before abstract mood language.
- Artist/song fingerprints should include non-confusion controls, especially for adjacent modern rap styles.
- Currentness should be separated from style mechanics: charts identify what matters now; production/vocal/songcraft analysis defines the usable fingerprint.
- Track style cards are the operational bridge between catalog profiles and MiniMax prompts.
