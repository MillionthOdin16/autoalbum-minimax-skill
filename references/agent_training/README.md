## Non-expert agent rule

If the agent cannot explain why an artifact improves the song or album using the terms in these references, the artifact is not ready. The agent must not rely on vague approvals such as “sounds good,” “has a vibe,” “is catchy,” “feels professional,” or “fits the genre.” It must identify the musical mechanism: hook, groove, contrast, prosody, arrangement, tension/release, vocal identity, lyric specificity, style fingerprint, production texture, sequence function, or MiniMax controllability.
