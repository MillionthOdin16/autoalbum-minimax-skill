# Arrangement, Form, and Energy Manual

This manual prevents the agent from making songs that are lyrically competent but musically flat.

## What arrangement does

Arrangement controls when the listener receives information and energy. A good arrangement creates expectation, contrast, payoff, and memory.

## Common forms

### Pop / R&B form
```text
[Intro] → [Verse] → [Pre Chorus] → [Chorus] → [Verse] → [Pre Chorus] → [Chorus] → [Bridge] → [Chorus] → [Outro]
```
Use when the song needs clear lift and replay.

### Rap / trap form
```text
[Intro] → [Hook] → [Verse] → [Hook] → [Verse] → [Hook] → [Outro]
```
or
```text
[Intro] → [Verse] → [Chorus] → [Verse] → [Chorus] → [Bridge/Break] → [Chorus]
```
Use hook-first when the hook is the main identity.

### Dance form
```text
[Intro] → [Build Up] → [Drop/Hook] → [Break] → [Build Up] → [Drop/Hook] → [Outro]
```
Use provider-supported tags only. If `[Drop]` is unsupported for the route, use `[Hook]`, `[Inst]`, or `[Build Up]` plus prompt instructions.

### Country/folk form
```text
[Verse] → [Chorus] → [Verse] → [Chorus] → [Bridge] → [Chorus] → [Outro]
```
Narrative clarity matters. Verse 2 must develop the story.

### Rock/pop-punk form
```text
[Intro] → [Verse] → [Pre Chorus] → [Chorus] → [Verse] → [Chorus] → [Bridge/Break] → [Chorus] → [Outro]
```
Use guitar energy, gang vocals, dynamic contrast, and drum lift.

## Energy mapping

Score each section 1-10.

Example:
```text
Intro: 3
Verse 1: 4
Pre Chorus: 6
Chorus: 8
Verse 2: 5
Bridge: 3 or 7 depending on strategy
Final Chorus: 9
Outro: 4
```

Rules:
- Not every song should only climb. Drops, resets, and negative space create impact.
- A chorus must change energy relative to the verse.
- A bridge must change something: harmony, density, perspective, vocal register, rhythm, or texture.
- The final chorus should justify its return: bigger, smaller, darker, more broken, more resolved, or meaningfully altered.

## Tension/release tools

- remove drums before hook
- introduce sub-bass at chorus
- shorten lines in pre-chorus
- add harmony layers at hook
- use a [Break] before final hook
- change chord color in bridge
- thin arrangement after dense section
- switch flow from dense to chant-like
- widen stereo field for chorus
- delay the downbeat or vocal entrance

## Adjacent-track contrast

For albums, two neighboring tracks should differ in at least two dimensions:

```text
tempo feel
groove
drum texture
bass behavior
vocal posture
hook type
harmonic color
arrangement shape
mix space
emotional function
```

## Arrangement failure modes

- every section has the same density
- chorus does not lift or change
- bridge is a third verse
- intro is too long without payoff
- outro fades without intention
- no space for MiniMax to create musical dynamics
- track has no identifiable peak
- album has repeated mid-tempo songs with similar drums

For every track, write:
