# Delegated Task: Quality Counterexample Review

## Objective
Compare the current artifact against known weak patterns and force revision when it resembles generic or non-professional output.

## Read
- `references/agent_training/quality_examples_and_counterexamples.md`
- `references/agent_training/agent_anti_shortcut_checklists.md`
- the current artifact under review
- relevant style/foundation/track references

## Process
1. Identify which artifact is being reviewed.
2. List three ways it could be mistaken for generic AI output.
3. Compare it to weak examples in the training file.
4. Identify specific lines, prompt sentences, sections, or decisions that need revision.
5. Recommend precise fixes.
6. Decide PASS, PASS_WITH_RISK, or FAIL_REVISION_REQUIRED.

## Output
Write to one of:

```text
tracks/tr_NN/quality_counterexample_review.md
reviews/quality_counterexample_review.md
```

## Rule
Do not pass an artifact because it is long, emotional, or coherent. Pass it only if it contains concrete musical mechanisms and would improve the final song or album.
