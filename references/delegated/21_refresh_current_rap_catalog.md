# Delegated: Refresh Current Rap Catalog

## Task

Refresh the current rap catalog before a major rap album generation run or when the user asks for the most current rap styles.

## Research targets

Use up-to-date chart/streaming/playlist/press signals, including:

```text
Billboard Hot Rap Songs
Billboard R&B/Hip-Hop charts
Apple Music U.S. Hip-Hop/Rap
Spotify U.S./Global rap playlists and top songs
major rap press and current reviews
platform playlist persistence
```

## Output files

```text
references/style_catalog/rap_current/current_popularity_basis_2026.md
references/style_catalog/rap_current/current_rap_refresh_2026_05_11.md
```

## For each candidate artist/song

Document:

```text
name
signal type
source/date
why it matters stylistically
whether it needs a new profile or only a song-reference card
adjacent profiles it may blur with
recommended action: add | elevate | keep | demote | ignore
```

## Quality bar

Do not add an artist just because they are popular. Add them if they contribute a distinct style lane or a current song-reference pattern useful for generation.
