# File Purpose Index

This skill intentionally keeps only four kinds of files:

1. **Activation/control files** — `SKILL.md`, `references/active/CURRENT.md`, architecture maps, controller JSON including phase applicability, and `scripts/autoalbum_guard.py`.
2. **Stage instructions** — delegated files under `references/delegated/` used by the canonical workflow.
3. **Reference knowledge** — agent training, craft, MiniMax API/provider docs, and style catalogs opened only when needed.
4. **Validation assets** — schemas and controller gate requirements used to verify artifacts.

Historical audits, superseded version maps, generated critic comments, and file inventories are intentionally excluded from the active package because they add context noise without improving execution.
