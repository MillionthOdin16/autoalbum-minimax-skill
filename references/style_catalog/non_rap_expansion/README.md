# Non-Rap Style Expansion

The current rap catalog is highly detailed. This directory raises the rest of the catalog closer to that standard. Use these files whenever the project is not rap-first or when rap is fused with pop, R&B, electronic, Latin/global, rock, country, or alternative production.

Each profile must translate user style requests into MiniMax-relevant musical controls:

```text
tempo / meter / groove
drum and percussion grammar
bass behavior
harmony / chord color
instrument palette
vocal delivery
hook mechanics
arrangement arc
mix and spatial texture
failure modes
adjacent-style differentiators
```

Do not approve a style card that only names a genre. The profile must make the generated output more likely to sound specific.
