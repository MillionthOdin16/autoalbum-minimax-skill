## Compilation rule

When fusing two artists, do not average them. Assign roles:

- **Beat fingerprint**: producer or genre source
- **Vocal fingerprint**: primary artist source
- **Hook fingerprint**: song/artist with highest appeal
- **Lyric posture**: album theme source
- **Mix space**: sonic world source

Example: `Future + Kendrick + Metro` should compile as:

- beat: Metro cinematic dark trap
- main hook/vibe: Future toxic melodic mantra
- feature/verse intensity: Kendrick competitive attack
- structure: one dramatic feature eruption

Not: generic dark trap with good lyrics.
