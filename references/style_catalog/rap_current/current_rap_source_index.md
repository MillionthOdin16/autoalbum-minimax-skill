# Current Rap Source Index

Use these sources when refreshing current rap style relevance. This index records source categories; refresh dates and exact chart positions should be updated during each project that depends on current taste.

## Chart and popularity sources
- Billboard Hot Rap Songs / Hot R&B/Hip-Hop Songs / Hot 100 / Billboard 200.
- Luminate year-end reports and Billboard/Luminate summaries.
- Apple Music Top Charts and Rap Life playlist.
- Spotify charts and major editorial playlists when accessible.
- YouTube Music charts and trending music videos when accessible.
- Kworb and Top-Charts mirrors when primary chart pages are inaccessible.

## Music press / style sources
- Pitchfork news/reviews for major rap album chart events and style commentary.
- Billboard artist and chart coverage.
- Complex, HotNewHipHop, Stereogum, HipHopDX, DJBooth, The Fader, and similar outlets for style/scene context.
- Producer/tutorial sources for drum, bass, and arrangement mechanics.

## Refresh protocol
For each current profile, record:
- source name,
- URL,
- access date,
- why it matters,
- whether it supports popularity, style mechanics, or both.

## Warning
Do not treat a chart snapshot as a full style analysis. Use charts to decide what matters now; use listening notes, production analysis, and profile differentiators to define the fingerprint.
